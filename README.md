# SQL Data Cleaning & Exploratory Data Analysis Project

## Overview
This project focuses on cleaning and analyzing a dataset using SQL.

## Project Structure
- data_cleaning.sql → Data cleaning process
- eda.sql → Exploratory Data Analysis

## Dataset
The dataset contains raw data with duplicates, missing values, and inconsistencies.

## Data Cleaning
- Removed duplicates using ROW_NUMBER()
- Standardized text values
- Handled NULL values
- Formatted dates correctly

## Exploratory Data Analysis (EDA)
- Analyzed total records
- Identified top industries
- Found trends over time
- Aggregated data using SUM and COUNT

## Tools Used
- MySQL Workbench
- SQL

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- SQL (CTEs, Window Functions, GROUP BY)
- Problem Solving
- ## Key Insights
- The tech industry recorded the highest number of layoffs, making it the most affected sector.
- Layoffs peaked in 2022, indicating a significant economic downturn during that period.
- A small number of companies accounted for a large proportion of total layoffs.
- The dataset required cleaning due to duplicates and missing values, highlighting common real-world data issues.
